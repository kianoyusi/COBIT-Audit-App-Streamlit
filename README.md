# Prototype Aplikasi Audit Tata Kelola TI dengan Python dan Streamlit

Proyek ini adalah purwarupa aplikasi web interaktif yang mensimulasikan proses evaluasi Tata Kelola TI berdasarkan 5 Domain COBIT (EDM, APO, BAI, DSS, MEA).

## Technical Stack & Fokus Engineering

* **Bahasa Pemrograman:** Python.
* **UI/Deployment Framework:** Streamlit (Digunakan untuk Logic Deployment dan UI Engineering).
* **Fokus Engineering:**
    1.  **Logic Deployment:** Mengubah logika perhitungan Python menjadi aplikasi yang dapat diakses melalui web.
    2.  **UI Engineering:** Implementasi menu navigasi sidebar dan input interaktif (Skala Likert) menggunakan Streamlit.
    3.  **Output:** Perhitungan skor rata-rata otomatis dan rekomendasi berdasarkan kriteria yang ditetapkan.

## Cara Menjalankan Aplikasi (Lokal)

1.  Pastikan Anda telah menginstal Python dan Streamlit: `pip install streamlit`
2.  Jalankan file kode melalui terminal: `streamlit run uts_app.py`
